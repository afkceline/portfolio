# SQL — Queries & Data Analysis

**Tools:** SQL (SQLite / standard SQL)  
**Focus:** Data extraction, transformation, and exploratory analysis

---

## Contents

| File | Description |
|------|-------------|
| `sql_mysql_libray.sql` | Basic queries on self-contained datasets: SELECT, WHERE, GROUP BY, ORDER BY, aggregate functions |
| `sqlite_and_pandas.ipynb` | Intermediate queries: JOINs, subqueries, filtering and data transformation |

Both files include the dataset creation statements — no external database needed. You can run them directly in any SQL environment (DB Browser for SQLite, DBeaver, etc.).

---

## Skills Demonstrated

- Writing clean, readable SQL queries
- Filtering, grouping, and aggregating data
- Joining multiple tables to answer business questions
- Structuring queries for exploratory data analysis

---

## How to Run

1. Open any SQL editor (DB Browser for SQLite recommended)
2. Open or paste the `.sql` file — datasets are created within the script
3. Run the queries step by step
